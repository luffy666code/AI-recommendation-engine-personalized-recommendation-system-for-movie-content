# 接口设计

## API 概述
基于 FastAPI 构建 RESTful API，提供电影推荐系统的后端服务。所有接口返回 JSON 格式数据。

## 基础信息
- 基础 URL：`http://localhost:8000`
- 认证：暂无（开发阶段）
- 数据格式：JSON

## 接口列表

### 电影相关接口
#### 1. 获取电影列表
- **方法**：GET
- **路径**：`/api/movies`
- **参数**：
  - `page` (可选): 页码，默认 1
  - `limit` (可选): 每页数量，默认 20
  - `genre` (可选): 电影类型筛选
  - `year` (可选): 年份筛选
- **响应**：
  ```json
  {
    "movies": [
      {
        "id": 1,
        "title": "Toy Story (1995)",
        "genres": ["Animation", "Children's", "Comedy"],
        "year": 1995
      }
    ],
    "total": 100,
    "page": 1,
    "pages": 5
  }
  ```

#### 2. 获取电影详情
- **方法**：GET
- **路径**：`/api/movies/{movie_id}`
- **参数**：movie_id (路径参数)
- **响应**：
  ```json
  {
    "id": 1,
    "title": "Toy Story (1995)",
    "genres": ["Animation", "Children's", "Comedy"],
    "year": 1995,
    "overview": "A cowboy doll is profoundly threatened and jealous...",
    "rating": 4.1,
    "rating_count": 452
  }
  ```

### 推荐相关接口
#### 3. 获取个性化推荐
- **方法**：GET
- **路径**：`/api/recommendations/{user_id}`
- **参数**：
  - `user_id` (路径参数): 用户 ID
  - `limit` (可选): 推荐数量，默认 10
- **响应**：
  ```json
  {
    "user_id": 1,
    "recommendations": [
      {
        "movie_id": 1,
        "title": "Toy Story (1995)",
        "score": 0.85,
        "reason": "基于您喜欢的动画电影推荐"
      }
    ]
  }
  ```

#### 4. 获取热门电影
- **方法**：GET
- **路径**：`/api/movies/popular`
- **参数**：
  - `limit` (可选): 返回数量，默认 10
- **响应**：
  ```json
  {
    "movies": [
      {
        "id": 1,
        "title": "Toy Story (1995)",
        "rating": 4.1,
        "rating_count": 452
      }
    ]
  }
  ```

### 用户相关接口
#### 5. 获取用户评分历史
- **方法**：GET
- **路径**：`/api/users/{user_id}/ratings`
- **参数**：user_id (路径参数)
- **响应**：
  ```json
  {
    "user_id": 1,
    "ratings": [
      {
        "movie_id": 1,
        "rating": 5.0,
        "timestamp": "1995-01-01T00:00:00Z"
      }
    ]
  }
  ```

## 错误响应格式
```json
{
  "error": "错误描述",
  "code": 400
}
```

## 状态码
- 200: 成功
- 400: 请求参数错误
- 404: 资源不存在
- 500: 服务器内部错误
