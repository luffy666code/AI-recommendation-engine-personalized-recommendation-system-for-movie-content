# 数据库设计

## 数据库概述
使用 SQLite 作为开发阶段的数据库，存储 MovieLens 数据集的电影、用户和评分信息。使用 SQLAlchemy 作为 ORM。

## 数据库配置
- 数据库类型：SQLite
- 数据库文件：`backend/app/db/movie_recommendation.db`
- ORM：SQLAlchemy

## 表结构设计

### 1. movies 表（电影表）
存储电影基本信息。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY | 电影 ID |
| title | VARCHAR(255) | NOT NULL | 电影标题 |
| genres | TEXT | NOT NULL | 电影类型（JSON 格式存储） |
| year | INTEGER |  | 上映年份 |
| overview | TEXT |  | 电影简介 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 更新时间 |

### 2. users 表（用户表）
存储用户信息。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY | 用户 ID |
| username | VARCHAR(100) | UNIQUE, NOT NULL | 用户名 |
| email | VARCHAR(255) | UNIQUE | 邮箱 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 更新时间 |

### 3. ratings 表（评分表）
存储用户对电影的评分记录。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY | 评分记录 ID |
| user_id | INTEGER | NOT NULL, FOREIGN KEY(users.id) | 用户 ID |
| movie_id | INTEGER | NOT NULL, FOREIGN KEY(movies.id) | 电影 ID |
| rating | FLOAT | NOT NULL | 评分值（1.0-5.0） |
| timestamp | DATETIME | DEFAULT CURRENT_TIMESTAMP | 评分时间 |

### 4. movie_stats 表（电影统计表）
存储电影的统计信息，用于推荐算法。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| movie_id | INTEGER | PRIMARY KEY, FOREIGN KEY(movies.id) | 电影 ID |
| avg_rating | FLOAT |  | 平均评分 |
| rating_count | INTEGER |  | 评分次数 |
| popularity_score | FLOAT |  | 流行度分数 |

## 索引设计
- movies 表：title, year, genres
- ratings 表：user_id, movie_id, rating
- movie_stats 表：popularity_score

## 数据关系
- users (1) -> ratings (N)
- movies (1) -> ratings (N)
- movies (1) -> movie_stats (1)

## 数据导入
- 从 MovieLens 数据集导入 movies.csv, ratings.csv, users.csv
- 计算并填充 movie_stats 表
- 使用 Pandas 进行数据预处理

## 注意事项
- SQLite 不支持外键约束的强制执行，但代码中会保持逻辑关系
- 评分数据量大时需要考虑性能优化
- 推荐算法会基于 ratings 表计算相似度矩阵
