# 项目规划

## 项目概述
基于数据挖掘的电影智能推荐系统，使用 MovieLens 数据集构建前后端分离的 Web 应用。后端负责数据处理、推荐算法和 API 服务，前端负责用户界面和交互。

## 项目目标
- 将原有 Jupyter Notebook 原型改造为可运行的 Web 系统
- 实现电影列表展示、电影详情查看、搜索筛选和个性化推荐功能
- 封装基于物品的协同过滤推荐算法
- 实现可运行、可展示、可扩展的本科毕业论文项目

## 技术栈
- **后端**：FastAPI
- **前端**：Vue 3 + Vite
- **数据库**：SQLite（开发阶段）
- **推荐算法**：基于物品的协同过滤 ItemCF
- **其他工具**：SQLAlchemy、Pandas、NumPy、scikit-learn

## 开发阶段
### 第一阶段：后端开发
- 初始化 FastAPI 项目结构
- 数据导入和数据库设计
- 推荐算法模块封装
- API 接口实现和测试

### 第二阶段：前端开发
- Vue 3 页面开发
- 前后端联调
- 用户界面优化

### 第三阶段：测试与优化
- 功能测试
- 性能优化
- 文档完善

## 注意事项
- 第一阶段只做后端开发，不涉及前端页面
- 统一使用 FastAPI，不使用 Flask
- 推荐算法专注于基于物品的协同过滤
- 数据库使用 SQLite 作为开发阶段的轻量级解决方案
